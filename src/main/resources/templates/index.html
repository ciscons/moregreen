프로젝트 목적

친환경적인 상품과 계획을 가지고 군중(Crowd)을 통해 소규모 후원을 받거나 인터넷과 같은 플랫폼을 통해 다수의 개인들 로부터 자금(Funding)을 모아서 누구나 쉽게 친환경이란 카테고리를 접하여 많은 친환경활동, 참여, 발전을 위한 친환경 크라우드펀딩 .

개발 과정

경기도 미래 기술학교에서 주최하는 클라우드 기반 JAVA 개발자 양성과정에서 파이널 프로젝트로 제작됨.
